# Library Management System (SQL)

This project is a relational database designed to manage a library system using SQL Server.

## Project Description

The database manages books, authors, members, staff, and book borrowing operations.  
It allows tracking borrowed books, overdue books, and analyzing borrowing activity.

## Database Tables

The system contains the following tables:

- Authors
- Categories
- Books
- Members
- Staff
- Loans

## Features

- Manage books and authors
- Organize books by categories
- Track library members
- Record borrowing and returning books
- Detect overdue books
- Analyze member activity

## Example Queries

The project includes several SQL queries such as:

- Books with authors and categories
- Active loans
- Overdue books
- Loans per member
- Loans processed by staff
- Top borrowed books
- Most active members
- Books per category

## Technologies Used

- SQL Server
- T-SQL
- Relational Database Design
